### git init - инициалиция локального репозитория
### git add . - добавляет индексацию(отслеживание) к файлам
### git push - позволяет отправить изменения в удалённый репозиторий
### Отличие локального репозитория от удалённого - локальный хранит данные на устройстве (доступ только у владельца), а удалённый на сервере (доступ у всей команды)
### Разница между папкой и репозиторием - репозиторий отслеживает изменения файлов и содержит внутри скрытую папку .git
### git commit ... - фиксирует текущее состояние изменений в истории локального репозитория после git add (то есть создаёт 
### "слепок" изменений для последующей отправки в удалённый репозиторий)
